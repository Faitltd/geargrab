# Background Images

Drop your images here: `src/lib/assets/backgrounds/`

## Requirements
- **Size**: 2560 x 1440 pixels
- **Format**: JPG
- **Content**: People doing outdoor activities (hiking, camping, biking, kayaking)
- **File size**: Under 1MB

Name them anything you want. The system will randomly use them as backgrounds.
